# ⚡ Void Walker Framework

**Void Walker Framework** là công cụ mô phỏng kiểm thử bảo mật và audit được viết bằng Python, tập trung vào:

- WiFi Security Audit
- Social Account Security Lab
- Server Security Lab
- Risk Analysis Engine
- Interactive Command Shell
- Security Simulation Modules

> Educational / Simulation Only — Không dùng cho mục đích trái phép.

---

## 🚀 Features

### WiFi Audit
- Scan mạng WiFi thực tế
- Phân tích mức độ rủi ro
- Rule-based security engine
- AI-style insight simulation

### Social Account Lab
- Phân tích rủi ro tài khoản mạng xã hội
- Attack/Defense simulation
- Privacy & security checks
- Report export

### Server Security Lab
- Reconnaissance simulation
- Service scanning
- Configuration audit
- Privilege escalation demo
- Risk assessment & reporting

### Command Modes
- Interactive terminal shell
- Role-based permissions
- Secure command mode
- Hidden security lab
